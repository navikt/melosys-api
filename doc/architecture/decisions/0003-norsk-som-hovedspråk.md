# 3. Norsk som hovedspråk

Date: ~3 år tilbake

## Status

Accepted

## Context

All kode i Melosys er skrevet på norsk. Dette innebærer at med unntak av ting som Java syntaks og metode prefiks som må være på engelsk, vil
alt annet være på norsk. Dette er en avgjørelse som ble tatt flere år tilbake og som ... @Francois

## Decision

Vi skal skrive all kode på norsk sålangt det er mulig.

## Consequences

På teamet i dag er dette noe omstridt. Mange ord og uttrykk i programmering er etablert på engelsk og det føles unaturlig å lage norske oversettelser.
Eksempler på dette er @RestControllere som postfikses med "Tjeneste", bruk av "er" istendefor "is" for boolske verdier.

Omfattende bruk av norsk krever også at man har tilgang på norskspråkelige utviklere. Dette er ikke et problem for øyeblikket, men man kan se
for seg at dette vil bli en utfordring senere. Innenfor programmering er engelsk "lingua franca" og det er uklart hvilken fordel det
skal være å bruke norsk. Dette vil bli tatt opp til ny vurdering, men foreløpig er det gjeldende.
